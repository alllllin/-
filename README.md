
## compare.py（模型选择）

`compare.py` 用于机器学习模型对比与筛选。

主要作用：
- 基于 `datest.csv` 的数据，分别针对两个目标进行建模：
  - `yield_strength_1000C`
  - `fracture_strain_RT`
- 对多种回归模型进行训练与评估。
- 通过指标（如 R2、RMSE、MAE）比较模型表现。
- 输出模型对比结果与图表，帮助确定后续优化阶段使用的模型（当前流程中使用 RF）。

## test2.py（迭代优化）

`test2.py` 用于多目标迭代搜索（优化）流程。

主要作用：
- 使用 RF + Bootstrap 估计预测均值与不确定性。
- 基于 EI（期望提升）构建双目标优化问题。
- 使用 NSGA-II 在约束成分空间内进行多轮独立搜索。
- 合并多个运行结果并提取全局帕累托前沿。
- 输出候选合金、全局帕累托结果和可视化图。
